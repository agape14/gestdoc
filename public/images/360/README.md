# Imágenes 360° para el Dashboard

Coloca aquí tus imágenes panorámicas 360° en formato equirectangular.

## Formatos soportados
- JPG/JPEG
- PNG

## Recomendaciones
- Resolución mínima: 4096x2048 píxeles
- Formato equirectangular (2:1 ratio)
- Tamaño de archivo optimizado (< 5MB para mejor rendimiento)

## Archivo predeterminado
El dashboard busca el archivo: `default-panorama.jpg`

Para cambiar la imagen, simplemente reemplaza este archivo o modifica la ruta en el componente Dashboard.

## Dónde conseguir imágenes 360°
- Toma fotos con una cámara 360° (Ricoh Theta, Insta360, etc.)
- Usa la app de 360 Photo Cam (https://360photocam.com/)
- Crea renders 360° con software 3D (Blender, 3ds Max, etc.)
